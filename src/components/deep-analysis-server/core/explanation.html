Interpretation Notes

This explanation is expressed in . Each contribution represents an additive change in that space.

When the explanation type is logit, contributions increase or reduce the model’s log-odds for the positive (MALWARE) class. The final probability is obtained by applying the logistic function to the summed logits.

When the explanation type is prob, contributions directly add or subtract probability mass for the positive class.
